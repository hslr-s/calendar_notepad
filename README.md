# 日历记事本(网页记事本)

#### 介绍
日历记事本是基于PHP语言的ThinkPHP5.1+前端Layui+日历插件进行开发的一个记事本软件。因为工作中常常要记录每天的工作日志，所以去年的上半年，抽了几天的下班时间开发了他，稳定运行了一年，但是他有些不足，现在决定重新启动此项目，对他进行优化和增加功能。希望给每个需要的人使用。
